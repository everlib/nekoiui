# Nekoi_UI

**@nekoi93/ui** — White flat modern glassmorphism design system built on shadcn/ui and Radix UI.

> 화이트 플랫 모던 글래시모피즘 / 3D 조명 버튼 / 일관된 그림자 깊이 / 플랫 메뉴 시스템

## 디자인 원칙

| 원칙 | 설명 |
|------|------|
| 화이트 플랫 모던 | 순수 흰색 기반, 과도한 색상 배제 |
| 글래시모피즘 | `backdrop-blur` + 반투명 배경 |
| 3D 버튼 조명 | top inset highlight + bottom shadow |
| 일관된 깊이 | CSS 변수로 정규화된 shadow 시스템 |
| 플랫 메뉴 | 배경 없는 underline / pill indicator |

---

## 컴포넌트

- **Button** — default / ghost / outline / accent / destructive / soft (6가지)
- **Card** — default / glass / float (3가지 surface)
- **Badge** — 8가지 의미론적 variant
- **Input** — inner-shadow depth + focus ring
- **Tabs** — underline / pill / card (3가지 스타일)
- **Select** — glass dropdown
- **Menu** — 수평 탑 내비 / pill 내비 / 수직 설정 메뉴 / glass 드롭다운 / 플로팅 도킹 바
